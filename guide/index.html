            # 契机

            在读小学的时候，语文课有个固定节目，是写日记。我曾经真的坚持了很长一段时间，写出来好多本，现在还放在书柜里。好的日记就像照片一样，是生活这场伟大实验的忠实记录者。当时我自然还没有明白这个道理，但是现在的我是很感激当时的自己。 我在读高中的时候，接触到了QQ。那是我接触的第一个社交软件，上面的朋友基本上都是同学，老师，学长学姐，学弟学妹。我的记录就搬到了QQ空间，也有不少朋友来点评。再后来的记录，或是在人人网，或是在360云盘，或是百度空间，移动硬盘。平台就如浪潮，后浪拍死前浪，而我的记录和前浪一起就那么消散了。至今还有不少照片在坏了的移动硬盘里读不出来，甚是可惜。 最早接触博客，是200x年了，当时博客还叫做部落格，新浪微博和腾讯微博还刚刚诞生。最早接触网页设计，也是200x年，当时大家都用flash，dreamweaver之类的工具，而且HTML5标准也没有出现。2016年左右，HTML5标准走入千家万户，internet explorer浏览器日薄西山，我重新学习了网页，而且还用自己的笔记本作为服务器，照猫画虎做了一个个人网站。可惜，由于没有舍得买ipv4地址和服务器，这个网站最终成了烂尾项目。 2019年，我留学回国，工作之余正有闲暇发展个人爱好，而且各种各样的框架任我挑选，我终于有机会做一回“后浪”，现代科技的成果被层层打开。B站(bilibili.com)的up主{飞跃高山与大洋的鱼}出了一套很棒的教程，本博客就是依照他的教程建立的。

            # 旅行

            19年的冬天，我刚刚入职半年，北京的一切都还很新鲜，我的日常就是工作日朝九晚六的上班，回家玩玩游戏看看书，周末出门浪。那个时候我对骑车有很高的热情，从矿大的学生手里弄来了一部二手的美利达 Duke500，周末到处瞎逛。从几十公里开始练腿，练到能连续骑上100公里。

            物极必反，我骑的有点太过了头，雾霾吸多了。去了医院，医生说我得了支气管炎。后面一直断断续续的咳嗽，总也不好。

            # 疫情

            2020是不平凡的一年，疫情让全球经济都受了影响。我最早关注到这个病毒实际上是在1月10号。那个时候有个公众号文章讲到这个病毒在一个医院里被发现，然后迅速完成了测序和上报。不过当时大家都没有意识到这个问题的严重程度，以为只是众多冠状病毒之中的一种。毕竟冠状病毒千千万，感染人类的大部分也都毒力不强。我当时的想法是CDC一定又过不好年了，政治压力下即使这个病毒很弱，也不敢怠慢的。

            到临近春节的时候，武汉的情况急转直下，出现了成千上万的患者。在疫情的冲击下，武汉的医院排起了长队，就诊秩序也开始混乱起来。全国人民似乎都在声讨武汉市长，卫健委，要追究他们隐瞒疫情的责任。人们回想起了03年的非典，却没有想到新冠比非典更可怕。虽然致死率不高，但是传染性和隐蔽性都大大增强。从疫情开始到国内基本控制住，总共感染近万人，死伤数千。

            不得不说，中国的政府和人民在这次疫情的表现还是让我有些惊叹。这种程度的配合和默契是建立在高度信任的基础上的，政府的大小官员多少也懂得这些道理，把舆论视为最高指示，凡是有报导出来在抗疫上出问题的领导和单位，处罚都是相当严厉的。

            北京的政策是春节放假延长到10号，从外地回京的居家隔离14天，然后去居委会领出入证，每天上下班凭证进出，还要测体温。我个人的感受是，这些措施的象征性意义大于实际作用。因为新冠患者或者携带者的体温即使有差别，手腕处在寒风中也测不出啥，但是只要这套程序还在做，人们就会觉得安心，也会保持秩序。

            关于疫情，我还有很多想掰扯的内容，可能单出一篇文章来写吧。

            # 亲朋好友

            19年的时候经常和朋友们约饭，我一度想要把我在北京的老同学全都请一遍，后来觉得人家和我也不一定有那么熟，何必尴尬。如果看到这里还没有被请过的在北京的朋友，请务必联系我哦~

            还有好多朋友，来北京找我叙旧，也是相谈甚欢。这个公开的博客我就不点名了，我还在研究如何用vuepress设置访问权限，以后会放在带权限的文章里来记录。

            自从来到北京，也有不少亲戚朋友给我介绍对象。2020年的8月，我认识了人生中第一位恋人，是一位朋友的朋友的朋友。可惜十月中旬又分开了。她离开的时候很淡漠，把我晾了好几天才肯亲口提分手，害我伤心了好久。

            终是塞上牛羊空许约，情深未变却寒盟。

            # 练字

            我刚回国的时候很多字都忘了怎么写，现在已经好很多了。虽然工作上终日敲键盘，但是手上拿笔写还是另一种思维模式。我最喜欢的钢笔是Lamy和Montblanc的，流畅顺滑，和铅笔一样可靠。我最喜欢的墨水是Noodler的Dark Matter。可能是受了小时候用鸵鸟蓝黑的影响。
